# Study of Memory Efficient Deep GNNs

## Project Proposal

In this project, we plan to study methods that allow training deeper Graph Neural Networks (GNNs) while reducing memory usage. The focus is on exploring techniques such as reversible architectures and weight sharing across layers.

## Team Members
- Kavya Gupta (22B1053)

## Objectives
- Understand the limitations of deep GNN models
- Study memory-efficient architectures for GNNs
- Implement baseline GNN models
- Compare performance and memory usage of different approaches

## Datasets
We plan to experiment with standard citation network datasets such as:
- Cora
- Citeseer
- PubMed

## Main Reference
Guohao Li et al. **"Training Graph Neural Networks with 1000 Layers"** [arXiv:2106.07476](https://arxiv.org/abs/2106.07476).
